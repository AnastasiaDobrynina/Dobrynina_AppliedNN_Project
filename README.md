# Упрощение академических текстов

**Учебный проект по курсу «Applied NN»**
**Автор:** Анастасия Добрынина 

## 📝 Описание проекта
Данный проект направлен на решение задачи автоматического упрощения сложных научных текстов (на материале лингвистических статей по синтаксису и морфологии) для восприятия школьниками 10-12 лет. 

Основная идея заключается в использовании подхода **Knowledge Distillation (Дистилляция знаний)**. Вместо использования тяжеловесных LLM для конечного вывода (Inference), знания передаются от мощной модели-учителя к компактной seq2seq модели-ученику, которая после дообучения способна выполнять задачу автономно, быстро и с минимальными вычислительными затратами.

---

## Данные
В проекте используется синтетический параллельный корпус текстов (около 450 пар), собранный в рамках этого же исследования.

* **Source (Входные данные):** Абзацы из реальных научных статей со сложной синтаксической структурой и академической лексикой.
* **Target (Псевдо-таргеты):** Те же абзацы, переписанные простым языком. Сгенерированы с помощью модели `Meta-Llama-3-8B-Instruct` через API Hugging Face. 
* **Препроцессинг:** Данные очищены от «галлюцинаций» и вежливых вступлений LLM (например, *"Here is the rewritten text:"*) с помощью регулярных выражений (RegEx), чтобы избежать заучивания моделью-учеником мусорных паттернов.

---

## 🧠 Краткое описание методов

1.  **Архитектура Ученика:** В качестве базовой модели выбрана `google/flan-t5-base` (~250 млн параметров). Архитектура Encoder-Decoder идеально подходит для задач трансформации текста.
2.  **Zero-shot Baseline:** Для оценки базового уровня `flan-t5-base` была протестирована «из коробки» без дообучения. Модель показала склонность к простому копированию исходного текста без его упрощения.
3.  **Fine-tuning (PEFT/LoRA):** Для дообучения модели на собранном корпусе использовался метод Low-Rank Adaptation (LoRA). 
    * Были заморожены основные веса модели, а обучались лишь небольшие матрицы-адаптеры ($r=8$, $\alpha=32$).
    * Это позволило избежать катастрофического забывания грамматики и провести обучение всего за несколько минут на мощностях Google Colab (T4 GPU).

## Инструкция по запуску

Весь код проекта содержится в едином Jupyter Notebook: `notebook.ipynb`.

1.  Склонируйте репозиторий:
    ```bash
    git clone [https://github.com/ВашЮзернейм/НазваниеРепозитория.git](https://github.com/ВашЮзернейм/НазваниеРепозитория.git)
    ```
2.  Откройте файл `notebook.ipynb` в **Google Colab** (рекомендуется для быстрого доступа к GPU).
3.  В левой панели Colab откройте вкладку **Secrets (Секреты)** и добавьте свой токен Hugging Face с именем `Colab_API` (необходимы права на `Make calls to Inference Providers`).
4.  Включите аппаратный ускоритель: `Среда выполнения` -> `Сменить среду выполнения` -> `T4 GPU`.
5.  Запускайте ячейки последовательно: от генерации датасета до обучения и инференса. 

*(Опционально: в папке `data/` лежат уже сгенерированные файлы `final_dataset_complete.csv` и `baseline_results.csv`, чтобы при желании можно было пропустить этап обращения к API и перейти сразу к обучению).*

## 📊 Результаты и выводы

**Вывод:**
Применение LoRA позволило успешно дистиллировать знания из Llama-3-8B в компактную T5-base. Дообученная модель перестала слепо копировать входной текст (как это делал Baseline) и научилась разбивать сложные синтаксические конструкции на более короткие и понятные предложения, заменяя научный жаргон нейтральной лексикой. Цель по созданию легковесного и эффективного упростителя текстов достигнута.
