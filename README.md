# Упрощение академических текстов

**Учебный проект по курсу «Applied NN»**
**Автор:** Анастасия Добрынина

## Описание проекта
Данный проект направлен на решение задачи автоматического упрощения сложных научных текстов (на материале лингвистических статей по синтаксису и морфологии) для восприятия школьниками 10-12 лет. 

Основная идея заключается в использовании подхода **Knowledge Distillation (Дистилляция знаний)**. Вместо использования тяжеловесных LLM для конечного вывода (Inference), знания передаются от мощной модели-учителя к компактной seq2seq модели-ученику, которая после дообучения способна выполнять задачу автономно, быстро и с минимальными вычислительными затратами.

**Проект охватывает следующие темы курса:**
1. Обработка естественного языка (NLP) и архитектуры Encoder-Decoder (Seq2Seq).
2. Параметрически эффективное дообучение больших языковых моделей (PEFT / LoRA).
3. Векторные представления текста (Embeddings) и семантический поиск (используются для оценки качества).

## Данные и структура репозитория
В проекте используется синтетический параллельный корпус текстов (~290 пар), собранный в рамках исследования из англоязычных научных статей в формате PDF.

* **Source (Входные данные):** Абзацы из реальных научных статей со сложной синтаксической структурой и академической лексикой.
* **Target (Псевдо-таргеты):** Те же абзацы, переписанные простым языком. Сгенерированы с помощью модели `Meta-Llama-3-8B-Instruct` в режиме zero-shot prompting через API Hugging Face. 
* **Препроцессинг:** Данные очищены от артефактов PDF (PyMuPDF), ссылочного аппарата и «галлюцинаций» LLM (например, вводных конструкций) с помощью регулярных выражений.

**Файлы в репозитории:**
* `Добрынина_проект_упрощение_текста.ipynb` — основной Jupyter Notebook с полным пайплайном проекта (сбор данных, бейзлайн, обучение, оценка).
* `data/final_dataset_complete.csv` — собранный и очищенный датасет.
* `data/final_metrics_evaluation.csv` — результаты генерации с рассчитанными метриками.
* `flan-t5-lora-simplifier/` — папка с сохраненными весами обученных LoRA-адаптеров (позволяет запустить инференс без повторного обучения).
* `AppliedNN_papers/` — папка с оригинальными сырыми данными (статьями).

## Краткое описание методов

1. **Архитектура Ученика:** В качестве базовой модели выбрана `google/flan-t5-base` (~250 млн параметров). 
2. **Zero-shot Baseline:** Для оценки базового уровня T5 была протестирована без дообучения. Модель показала склонность к простому копированию исходного текста без изменения синтаксиса.
3. **Fine-tuning (PEFT/LoRA):** Для дообучения модели использовался метод Low-Rank Adaptation. Были заморожены основные веса модели, обучались лишь небольшие матрицы-адаптеры ($r=8$, $\alpha=32$). Это предотвратило катастрофическое забывание и позволило провести обучение за 3 минуты на T4 GPU.
4. **Оценка качества:** Использовались две независимые метрики:
   * **ROUGE:** для оценки лексического совпадения с псевдо-таргетом (навык копирования стиля учителя).
   * **Косинусное сходство эмбеддингов (SentenceTransformers):** для оценки сохранения фактологии при сравнении оригинального научного текста с финальным упрощением.

## Инструкция по запуску

Для запуска проекта рекомендуется использовать **Google Colab**.

1. Склонируйте репозиторий:
```bash
git clone https://github.com/ВашЮзернейм/НазваниеРепозитория.git

```


2. Откройте файл `Добрынина_проект_упрощение_текста.ipynb` в Google Colab.


3. Включите аппаратный ускоритель: `Среда выполнения` -> `Сменить среду выполнения` -> `T4 GPU` (потребуется только для этапа дообучения, инференс работает и на CPU).
4. В левой панели Colab откройте вкладку **Secrets (Секреты)** и добавьте свой токен Hugging Face с именем `Colab_API` (необходим для шага обращения к API).
5. Запускайте ячейки последовательно.
*Примечание: Вы можете пропустить долгие этапы сбора данных и обучения, загрузив готовые датафреймы из папки `data/` и веса LoRA-адаптеров с помощью соответствующей ячейки в разделе Inference.*

## Результаты и выводы

Применение LoRA позволило успешно дистиллировать знания из Llama-3-8B в компактную T5-base. Дообученная модель преодолела baseline (которому было свойственно слепое копирование исходного текста) и научилась разбивать сложные синтаксические конструкции на короткие и понятные предложения.

**Анализ метрик и ошибок:**

* Метрики ROUGE (ROUGE-L: 0.1392) и среднее косинусное сходство (0.4419) демонстрируют умеренный уровень совпадения. Это является нормой для задачи глубокого перефразирования, так как при упрощении полностью меняется структура предложений и удаляется сложная терминология.
* В ходе качественного анализа ошибок была выявлена проблема «Prompt Leakage»: при подаче на вход шумовых данных (номера сносок, фрагменты библиографии) модель возвращает текст системного промпта. На очищенных лингвистических данных модель успешно справляется с задачей, теряя контекст лишь в случаях избыточно длинных сложноподчиненных предложений.

**Итог:** Цель по созданию легковесного и эффективного упростителя академических текстов успешно достигнута.
